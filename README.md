# Personal Library Manager CLI

A Command-Line Interface (CLI) application for managing a personal library of books, authors, and genres using Python and SQLAlchemy ORM.  

This project demonstrates the use of a relational database with multiple related tables, proper package structure, and a fully functional CLI built with Click.

## Features
- Add, list, and manage **books**, **authors**, and **genres**
- Relational database with **three tables** (`books`, `authors`, `genres`)
- Fully functional CLI interface with **Click**
- Database schema managed with **Alembic migrations**
- Sample data population script included
